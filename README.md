## 📝 Описание программы

Expense Tracker — это графическое приложение для отслеживания личных расходов. Программа позволяет:

- Добавлять расходы с указанием суммы, категории и даты
- Фильтровать расходы по категории и периоду
- Подсчитывать общую сумму расходов и сумму за выбранный период
- Сохранять и загружать данные в формате JSON
- Удалять ненужные записи
